# mini-games

- `word-spelling-practice-duck/word-spelling-practice-duck.html`：小鸭子单词拼写练习，听提示、选字母，练习拼单词。
- `duck-letter-hop/duck-letter-hop.html`：小鸭子字母跳跳练习，边跳边认字母和练反应。
- `cloud-bakery-hop/cloud-bakery-hop.html`：云朵、金币、面包店和钻土躲刺球的自由小游戏。
- `duck-market-hop/duck-market-hop.html`：小鸭子的跳跳日常，出门买菜、回家做家务、过小河去厕所并跳起躲小气球。
- `xiao-yaya-daily/xiao-yaya-daily.html`：小丫丫换衣、买菜、自动安排晚饭和拜访小浣熊家的日常小游戏。
